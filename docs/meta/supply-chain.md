# Supply Chain

Release artifacts are published through GitHub Releases. Each tagged release includes the source tarball and a CycloneDX SBOM artifact.

- Latest release assets: <https://github.com/lreading/slide-spec/releases/latest>
- The release asset list includes `sbom.cyclonedx.json` on tagged releases that publish an SBOM.

SBOM details and access notes are documented under [Meta > SBOM](/meta/sbom).
